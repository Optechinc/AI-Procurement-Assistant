# AI Procurement Assistant

An AI-powered procurement assistant that helps procurement professionals analyze Request for Quotations (RFQs), summarize solicitation documents, extract key requirements, and generate supplier communications using Large Language Models (LLMs).

## Features

- Upload RFQ documents
- Extract solicitation requirements
- Generate supplier emails
- Create compliance checklists
- Summarize RFQs with AI

## Tech Stack

- Python
- Git
- GitHub
- Streamlit
- OpenAI API
